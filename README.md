# @gubareve/xsquared

Removes all spaces from a string.

## Install

```
$ npm install @gubareve/xsquared
```

## Usage

```js
const xsquared = require("@gubareve/xsquared");

xsquared(4);
//=> "16

xsquared("4");
//=> Uncaught TypeError: xsquared requires a integer
//    at xsquared (<anonymous>:2:41)
//    at <anonymous>:1:1
```
